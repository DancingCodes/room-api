# Room 第一版开发任务拆分

## 1. 后端任务

### 1.1 项目初始化

- 初始化 Go module：`room-api`
- 搭建 Gin 服务
- 接入 `.env`
- 接入统一响应
- 接入 CORS
- 实现 `/health`

### 1.2 数据库

- 接入 GORM
- 接入 MySQL 云数据库
- 实现 AutoMigrate
- 定义 users
- 定义 email_verification_codes
- 定义 rooms
- 定义 room_members
- 定义 messages

### 1.3 用户认证

- bcrypt 密码哈希
- JWT 生成和解析
- JWT 中间件
- 注册邮箱验证码
- 注册
- 登录
- 当前用户信息
- 修改昵称
- 重置密码验证码
- 重置密码

### 1.4 文件上传

- 腾讯云 COS 配置
- 注册前头像上传
- 登录后头像更新
- 文件类型校验
- 文件大小校验

### 1.5 房间

- 创建房间
- 房间列表
- 房间详情
- 进入房间
- 离开房间
- 房主自动转让
- 最后一人离开后删除房间和消息

### 1.6 消息

- 历史消息
- 发送文字消息
- 消息长度校验
- WebSocket 广播

### 1.7 WebSocket

- 房间连接池
- 鉴权
- 心跳
- 断开后离开房间
- 成员加入广播
- 成员离开广播
- 房主变更广播
- 消息广播
- 麦克风状态广播

## 2. Android 任务

- 创建 Android 项目
- 配置 Kotlin / Jetpack
- 配置 Retrofit / OkHttp
- 配置 DataStore
- 配置 Coil
- 登录页
- 注册页
- 忘记密码页
- 房间列表页
- 创建房间弹窗
- 房间详情页
- 个人资料页
- HTTP 请求封装
- 图片选择和上传
- WebSocket 管理
- 表单校验
- Toast / Loading

## 3. 联调任务

- 注册验证码
- 注册前头像上传
- 注册
- 登录
- 修改头像
- 修改昵称
- 创建房间
- 房间列表
- 进入房间
- 两个 Android 客户端实时聊天
- 开麦/闭麦状态同步
- 房主离开后转让
- WebSocket 断开后自动离开

## 4. 验收任务

- 按 [测试用例.md](./测试用例.md) 完成主流程测试
- 检查云数据库数据
- 检查 COS 文件上传
- 检查 QQ 邮箱验证码
- 检查 Android 真机体验
