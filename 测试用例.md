# Room 第一版测试用例

## 1. 用户注册

- 头像为空，注册失败
- 用户名格式错误，注册失败
- 邮箱格式错误，注册失败
- 昵称为空，注册失败
- 昵称超过 8 位，注册失败
- 密码少于 6 位，注册失败
- 邮箱验证码错误，注册失败
- 邮箱验证码过期，注册失败
- 注册成功后返回 token
- 注册成功后可获取当前用户信息

## 2. 登录

- 用户名为空，登录失败
- 密码为空，登录失败
- 密码错误，登录失败
- 登录成功后返回 token
- token 过期后请求返回 401

## 3. 邮箱验证码

- 注册验证码发送成功
- 60 秒内重复发送失败
- 1 小时超过 5 次发送失败
- 重发验证码后旧验证码失效
- 重置密码验证码发送成功
- 邮箱发送失败时返回 500

## 4. 头像上传

- jpg 上传成功
- png 上传成功
- webp 上传成功
- 非图片上传失败
- 超过 2MB 上传失败
- COS 上传失败时返回 500
- 上传成功后返回 COS URL

## 5. 个人资料

- 修改昵称成功
- 修改为重复昵称失败
- 修改头像成功
- 用户名不可修改
- 邮箱不可修改

## 6. 房间

- 创建 2 人房成功
- 创建 8 人房成功
- 创建非法人数房间失败
- 创建房间后用户自动成为房主
- 房间名称为 `昵称 + 的房间`
- 房间列表按创建时间倒序
- 房间满员后不能进入
- 一个用户不能同时进入两个房间

## 7. 房主转让

- 非房主离开，房主不变
- 房主离开且还有成员，最早加入的成员成为新房主
- 最后一个成员离开，房间删除
- 最后一个成员离开，房间消息删除

## 8. 消息

- 发送空消息失败
- 发送 trim 后为空的消息失败
- 发送超过 50 字消息失败
- 发送正常消息成功
- 另一个用户通过 WebSocket 收到消息
- 历史消息可以分页获取
- 历史消息返回顺序为 id 正序

## 9. 麦克风状态

- 开麦成功
- 闭麦成功
- 非房间成员更新麦克风失败
- 房间其他成员收到麦克风状态广播

## 10. WebSocket

- token 正确时连接成功
- token 错误时连接失败
- 成员加入时收到 `member.joined`
- 成员离开时收到 `member.left`
- 房主变更时收到 `room.owner_changed`
- 消息发送时收到 `message.created`
- 麦克风状态变化时收到 `member.mic_updated`
- WebSocket 断开后用户离开房间
