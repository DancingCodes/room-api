# Room 环境变量说明

## 1. 服务

```txt
APP_PORT=8080
```

后端服务端口。

```txt
CORS_ALLOW_ORIGINS=*
```

CORS 允许来源。开发期可以使用 `*`，上线后建议配置为实际 App/API 域名。

## 2. JWT

```txt
JWT_SECRET=change_me
```

JWT 签名密钥。上线环境必须使用强随机字符串。

## 3. MySQL

```txt
MYSQL_DSN=room:password@tcp(127.0.0.1:3306)/room?charset=utf8mb4&parseTime=True&loc=Local
```

云数据库连接串。

要求：

- 云数据库提前创建 database：`room`
- 服务启动后执行 GORM AutoMigrate
- 连接失败时服务启动失败并退出

## 4. QQ 邮箱 SMTP

```txt
SMTP_HOST=smtp.qq.com
SMTP_PORT=465
SMTP_USERNAME=your_qq_number@qq.com
SMTP_PASSWORD=your_qq_email_authorization_code
SMTP_FROM=your_qq_number@qq.com
```

说明：

- `SMTP_PASSWORD` 使用 QQ 邮箱授权码
- 不使用 QQ 登录密码
- 邮件发送失败时接口返回 `500`

## 5. 腾讯云 COS

```txt
COS_SECRET_ID=your_tencent_cloud_secret_id
COS_SECRET_KEY=your_tencent_cloud_secret_key
COS_REGION=ap-guangzhou
COS_BUCKET=your_bucket_name
COS_BASE_URL=https://your_bucket.cos.ap-guangzhou.myqcloud.com
```

说明：

- 头像文件公开读
- 文件路径统一为 `room/{uuid}.{ext}`
- COS 上传失败时接口返回 `500`
